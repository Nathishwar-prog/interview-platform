The Interview Preparation Platform is a modern, scalable web application designed to assist users in preparing for technical interviews. It offers a company-wise approach, allowing users to access curated interview questions, coding problems, and real interview experiences from various tech companies. The platform aims to provide a comprehensive resource for job seekers, enhancing their chances of success in interviews across different roles and companies.

Project Module Description
The platform consists of several core modules:

Home Page: Displays featured companies, trending questions, and a global search.
Companies Page: Lists companies with filters for difficulty, type, and hiring type.
Company Detail Page: Provides in-depth information about a specific company, including interview processes and required skills.
Question Detail Page: Displays details about specific interview questions, including expected answers and tips.
Problem Detail Page: Offers coding problems with statements, examples, and solutions.
Bookmarks Page: Allows users to save and manage their favorite questions and problems.

Usage
To get started with the project, follow these steps:

Install dependencies:
pnpm install
Build the application:
pnpm run build
Run the application:
pnpm run start
